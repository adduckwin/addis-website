## 1. Общая информация о проекте

### 1.1 Описание
- **Название**: Addis Coffee
- **Тип**: E-commerce сайт для обжарочного производства кофе
- **Целевая аудитория**: Розничные клиенты, HoReCa, оптовые покупатели
- **Текущий стек**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui

### 1.2 Структура проекта
```
app/
├── src/
│   ├── components/        # UI компоненты (shadcn/ui)
│   │   ├── ui/           # Базовые компоненты (Button, Input, Card, etc.)
│   │   └── cart/         # Компоненты корзины
│   ├── data/             # Статические данные
│   │   ├── products.ts   # 50 товаров
│   │   └── blog.ts       # 15 статей блога
│   ├── pages/            # Страницы приложения
│   ├── sections/         # Секции главной страницы
│   ├── store/            # Zustand store (корзина, профиль)
│   ├── types/            # TypeScript типы
│   └── lib/              # Утилиты
├── public/               # Статические файлы
└── dist/                 # Сборка
```

## 7. Бонусная программа (Лояльность)

### 7.1 Начисление баллов
| Действие | Баллы | Условия |
|----------|-------|---------|
| Покупка | 1 балл за 100 ₽ | После доставки заказа |
| Заполнение профиля | +50 | Дата рождения |
| Заполнение профиля | +30 | Пол |
| Заполнение профиля | +40 | Предпочтения |
| Регистрация | +20 | Телефон |

### 7.2 Использование баллов
- 1 балл = 1 рубль
- Максимум 50% от суммы заказа
- Не применяется к доставке

### 7.3 Реализация в Битрикс
```php
// Начисление бонусов при смене статуса заказа
AddEventHandler("sale", "OnOrderStatusSendEmail", "AddBonusPoints");

function AddBonusPoints($orderID, &$eventName, &$arFields) {
    $order = CSaleOrder::GetByID($orderID);
    if ($arFields["STATUS_ID"] == "F") { // Заказ выполнен
        $bonusPoints = floor($order["PRICE"] / 100);
        $userId = $order["USER_ID"];
        // Начислить баллы пользователю
        $user = new CUser;
        $user->Update($userId, [
            "UF_BONUS_POINTS" => $currentBonus + $bonusPoints
        ]);
    }
}
```

## 8. Оптовые цены и скидки

### 8.1 Уровни скидок
| Уровень | Название | Скидка | Мин. заказ/мес |
|---------|----------|--------|----------------|
| 1 | Стартовый | 15% | 50 000 ₽ |
| 2 | Базовый | 20% | 100 000 ₽ |
| 3 | Продвинутый | 25% | 200 000 ₽ |
| 4 | Партнёрский | 30% | 500 000 ₽ |

### 8.2 Реализация в Битрикс
```php
// Применение скидки в зависимости от группы пользователя
function ApplyWholesaleDiscount($price, $userId) {
    $userGroups = CUser::GetUserGroup($userId);
    
    $discounts = [
        WHOLESALE_TIER_1 => 0.15,
        WHOLESALE_TIER_2 => 0.20,
        WHOLESALE_TIER_3 => 0.25,
        WHOLESALE_TIER_4 => 0.30,
    ];
    
    foreach ($discounts as $groupId => $discount) {
        if (in_array($groupId, $userGroups)) {
            return $price * (1 - $discount);
        }
    }
    
    return $price; // Розничная цена
}
```
