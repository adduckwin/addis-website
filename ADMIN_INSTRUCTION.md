## 1. Начало работы

### 1.1. Вход в админ-панель
1. Откройте браузер и перейдите по адресу: `https://addis-coffee.ru/admin-login`
2. Введите логин и пароль, полученные от разработчика
3. После успешного входа вы попадете на главный дашборд

### 1.2. Обзор дашборда
На главной панели отображаются:
- **Статистика продаж** за сегодня/неделю/месяц
- **Последние заказы** со статусами
- **Новые зарегистрированные клиенты**
- **Остатки товаров** (требующие пополнения)
- **Быстрые действия**: добавить товар, создать заказ, написать в блог

## 2. Управление товарами

### 2.1. Просмотр каталога
1. Перейдите в раздел **Товары** → **Каталог**
2. Используйте фильтры:
   - По категории (Зерновой, Молотый, Капсулы)
   - По наличию (В наличии / Нет на складе)
   - По цене
3. Список товаров отображается в виде таблицы с возможностью сортировки

### 2.2. Добавление нового товара
1. Нажмите кнопку **+ Добавить товар**
2. Заполните обязательные поля:
   - **Название** (например, "Эфиопия Иргачефф")
   - **Артикул** (уникальный код, например, ETH-001)
   - **Категория** (выберите из списка)
   - **Цена розничная** (₽)
   - **Цена оптовая** (₽, автоматически применяется при заказе от 5 кг)
   - **Остаток** (кг)

3. Заполните дополнительные поля:
   - **Описание**: краткое описание вкуса (до 500 символов)
   - **Характеристики**:
     - Страна происхождения
     - Регион выращивания
     - Высота произрастания
     - Способ обработки (мытая, натуральная, хани)
     - Степень обжарки (светлая, средняя, темная)
     - Вкусовые ноты (выберите из тегов или впишите свои)
   - **Фасовка**: выберите доступные варианты (250г, 500г, 1кг)

4. Загрузите изображения:
   - **Основное фото** (обязательно, мин. 800x800px)
   - **Дополнительные фото** (до 5 штук)
   - Рекомендуется использовать формат JPG/PNG

5. Настройте SEO:
   - **Meta Title** (до 60 символов)
   - **Meta Description** (до 160 символов)
   - **URL-алиас** (автоматически генерируется из названия)

6. Нажмите **Сохранить**

### 2.3. Редактирование товара
1. Найдите товар в каталоге
2. Нажмите на иконку ✏️ (Редактировать)
3. Внесите изменения
4. Нажмите **Сохранить изменения**

### 2.4. Управление остатками
**Вариант 1: Ручное обновление**
1. В списке товаров нажмите на поле "Остаток"
2. Введите новое значение
3. Изменение сохранится автоматически

**Вариант 2: Массовое обновление**
1. Отметьте товары чекбоксами
2. Выберите действие **Изменить остатки**
3. Укажите тип операции:
   - **Установить значение** (заменить текущее)
   - **Прибавить** (увеличить на N кг)
   - **Вычесть** (уменьшить на N кг)
4. Нажмите **Применить**

**Вариант 3: Синхронизация со СБИС**
1. Перейдите в **Настройки** → **Интеграции**
2. Нажмите **Синхронизировать товары**
3. Система загрузит актуальные остатки из СБИС
4. Проверьте отчет об ошибках (если есть)

### 2.5. Управление категориями
1. Перейдите в **Товары** → **Категории**
2. Для добавления категории:
   - Нажмите **+ Добавить категорию**
   - Укажите название (например, "Для эспрессо")
   - Загрузите изображение-превью
   - Выберите родительскую категорию (если есть)
   - Настройте порядок отображения (сортировка)
3. Сохраните изменения

## 3. Управление заказами

### 3.1. Просмотр заказов
1. Перейдите в раздел **Заказы**
2. Используйте фильтры:
   - По статусу (Новый, В работе, Отправлен, Выполнен, Отменен)
   - По типу (Розница / Опт)
   - По дате создания
   - По клиенту

### 3.2. Детали заказа
При клике на номер заказа открывается карточка с информацией:
- **Данные клиента**: ФИО, телефон, email, адрес доставки
- **Состав заказа**: товары, количество, цены, скидки
- **Сумма**: стоимость товаров, доставка, итоговая сумма
- **Статус оплаты**: Оплачено / Не оплачено / Частично
- **История изменений**: кто и когда менял статус

### 3.3. Обработка нового заказа
**Для розничных заказов:**
1. Проверьте наличие товаров на складе
2. Подтвердите заказ (статус: **В работе**)
3. Распечатайте накладную (кнопка **Печать**)
4. Передайте заказ в службу доставки
5. Обновите статус на **Отправлен**
6. После получения клиентом установите статус **Выполнен**

**Для оптовых заказов:**
1. Проверьте статус компании (ИНН должен быть верифицирован)
2. Проверьте соответствие оптовым ценам
3. Сформируйте счет на оплату (кнопка **Сформировать счет**)
4. Отправьте счет клиенту (автоматически через email или вручную)
5. После оплаты обновите статус на **Оплачен**
6. Создайте документы для отгрузки
7. Передайте в отдел логистики

### 3.4. Изменение статуса заказа
1. Откройте заказ
2. В блоке "Статус" выберите новое значение из списка
3. При необходимости добавьте комментарий (виден клиенту)
4. Нажмите **Сохранить**
5. Клиент получит уведомление об изменении статуса (email/SMS)

### 3.5. Возвраты и отмены
1. Откройте заказ
2. Нажмите **Оформить возврат**
3. Укажите причину возврата:
   - Брак товара
   - Ошибка комплектации
   - Отказ клиента
   - Другое
4. Выберите товары для возврата
5. Укажите сумму к возврату
6. Сохраните заявку
7. Статус заказа изменится на **Возврат**

## 4. Работа с клиентами

### 4.1. Просмотр клиентов
1. Перейдите в раздел **Клиенты**
2. Фильтры:
   - По типу (Розничный / Оптовый)
   - По статусу (Активен / Заблокирован)
   - По дате регистрации

### 4.2. Карточка клиента
При клике на клиента открывается подробная информация:
- **Личные данные**: ФИО, контакты
- **Компания** (для оптовиков): ИНН, КПП, юридический адрес
- **История заказов**: список всех заказов с суммами
- **Бонусные баллы**: начислено / потрачено / доступно
- **Персональные скидки** (если установлены)

### 4.3. Одобрение оптового клиента
1. Новые заявки на оптовую регистрацию появляются со статусом **На проверке**
2. Проверьте данные компании:
   - ИНН должен быть верифицирован через DaData
   - Компания должна быть действующей
3. Если все корректно:
   - Нажмите **Одобрить**
   - Клиент получит доступ к оптовым ценам
   - Статус изменится на **Оптовый клиент**
4. Если есть проблемы:
   - Нажмите **Отклонить**
   - Укажите причину (видна клиенту)
   - Клиент получит уведомление

### 4.4. Ручное создание клиента
1. Нажмите **+ Добавить клиента**
2. Выберите тип (Розничный / Оптовый)
3. Заполните данные:
   - Для розничного: ФИО, телефон, email, адрес
   - Для оптового: реквизиты компании (можно ввести ИНН для автозаполнения)
4. Установите пароль или отправьте приглашение на email
5. Сохраните

### 4.5. Управление бонусными баллами
1. Откройте карточку клиента
2. В разделе "Бонусы" нажмите **Начислить баллы** или **Списать баллы**
3. Укажите количество баллов и причину
4. Добавьте комментарий (опционально)
5. Сохраните

## 5. Управление контентом

### 5.1. Блог

#### Создание статьи
1. Перейдите в **Контент** → **Блог**
2. Нажмите **+ Добавить статью**
3. Заполните поля:
   - **Заголовок** (H1, до 100 символов)
   - **Краткое описание** (анонс для превью, до 200 символов)
   - **Текст статьи**: используйте визуальный редактор
     - Форматирование текста (жирный, курсив, списки)
     - Вставка изображений (перетаскиванием или через кнопку)
     - Добавление видео (YouTube/Vimeo ссылки)
     - Вставка цитат и кода
   - **Изображение превью** (для карточки в списке)
   - **SEO-параметры**:
     - Meta Title
     - Meta Description
     - URL-алиас (например, `/blog/kak-pravilno-zavarivat-kofe`)
   - **Категория** (Новости, Советы, Рецепты)
   - **Теги** (через запятую)
4. Сохраните как черновик или опубликуйте сразу

#### Редактирование статьи
1. Найдите статью в списке
2. Нажмите ✏️ (Редактировать)
3. Внесите изменения
4. Сохраните

#### Публикация/Снятие с публикации
- Переключатель **Статус** позволяет быстро публиковать или скрывать статью
- Черновики видны только администраторам
- Опубликованные статьи доступны всем посетителям

### 5.2. Баннеры на главной странице
1. Перейдите в **Контент** → **Баннеры**
2. Для добавления баннера:
   - Нажмите **+ Добавить баннер**
   - Загрузите изображение (рекомендуемый размер: 1920x600px)
   - Укажите заголовок и подзаголовок
   - Добавьте ссылку (на категорию, товар или статью)
   - Настройте порядок отображения
   - Выберите период показа (дата начала и окончания)
3. Сохраните

### 5.3. Страницы
Для редактирования статических страниц (О нас, Доставка, Контакты):
1. Перейдите в **Контент** → **Страницы**
2. Выберите нужную страницу
3. Отредактируйте текст в визуальном редакторе
4. Сохраните изменения

## 6. Настройка дизайна

### 6.1. Доступ к настройкам дизайна
1. Перейдите в **Настройки** → **Дизайн**
2. Откроется панель с предпросмотром сайта

### 6.2. Изменение цветовой схемы
1. В разделе **Цвета** выберите элемент для настройки:
   - **Основной цвет** (кнопки, ссылки, акценты)
   - **Вторичный цвет** (фон секций, плашки)
   - **Цвет акцента** (special offers, badges)
   - **Цвет текста** (основной, вторичный)
   - **Цвет фона** (светлый, темный)

2. Используйте Color Picker:
   - Кликните на цветовое поле
   - Выберите цвет визуально или введите HEX/RGB код
   - Предпросмотр обновляется в реальном времени

3. Нажмите **Применить** для сохранения

### 6.3. Настройка типографики
1. В разделе **Шрифты** выберите:
   - **Шрифт заголовков** (H1-H6)
   - **Шрифт основного текста**
   
2. Доступные шрифты подгружаются из Google Fonts:
   - Roboto
   - Open Sans
   - Montserrat
   - Playfair Display
   - И другие (полный список в dropdown)

3. Настройте размеры:
   - Размер заголовка H1 (по умолчанию 48px)
   - Размер заголовка H2 (по умолчанию 36px)
   - Размер основного текста (по умолчанию 16px)

4. Сохраните изменения

### 6.4. Скругления элементов
1. В разделе **Формы** используйте слайдер **Радиус скругления**
2. Диапазон: от 0px (острые углы) до 20px (сильное скругление)
3. Изменение применяется к:
   - Кнопкам
   - Карточкам товаров
   - Полям ввода
   - Модальным окнам

### 6.5. Логотипы
1. В разделе **Логотипы** загрузите файлы:
   - **Логотип для хедера** (SVG/PNG, макс. 200x60px)
   - **Логотип для футера** (SVG/PNG, макс. 200x60px)
   - **Favicon** (ICO/PNG, 32x32px)

2. Рекомендуемые форматы:
   - SVG для векторных логотипов (лучшее качество)
   - PNG с прозрачным фоном

3. После загрузки нажмите **Сохранить**

### 6.6. Сброс к дефолтным настройкам
Если нужно вернуть исходный дизайн:
1. Нажмите кнопку **Сбросить к заводским настройкам**
2. Подтвердите действие во всплывающем окне
3. Все изменения будут отменены

### 6.7. Публикация изменений
- Изменения применяются мгновенно в режиме предпросмотра
- Для применения на боевом сайте нажмите **Опубликовать изменения**
- Все пользователи увидят новый дизайн после обновления страницы

## 7. Настройка интеграций

### 7.1. Интеграция со СБИС

#### Первоначальная настройка
1. Перейдите в **Настройки** → **Интеграции** → **СБИС**
2. Получите доступы к API СБИС:
   - Зарегистрируйтесь в личном кабинете СБИС
   - Создайте приложение в разделе "Разработчикам"
   - Получите `App ID` и `App Secret`
   
3. Введите данные в форму:
   - **API URL**: `https://api.sbistenant.ru` (или ваш тенант)
   - **App ID**: ваш идентификатор приложения
   - **App Secret**: ваш секретный ключ
   - **ID организации**: ID вашей компании в СБИС

4. Нажмите **Проверить соединение**
   - При успешной проверке появится зеленая галочка
   - При ошибке проверьте правильность введенных данных

5. Сохраните настройки

#### Синхронизация товаров
1. В разделе **СБИС** нажмите **Синхронизировать товары**
2. Выберите параметры:
   - **Обновлять цены**: да/нет
   - **Обновлять остатки**: да/нет
   - **Создавать новые товары**: да/нет
   - **Архивировать отсутствующие**: да/нет

3. Запустите синхронизацию
4. Дождитесь завершения (может занять несколько минут)
5. Проверьте отчет:
   - Обновлено товаров: X
   - Создано новых: Y
   - Ошибки: Z (если есть)

#### Автоматическая синхронизация
Для настройки расписания:
1. Включите переключатель **Автоматическая синхронизация**
2. Выберите интервал:
   - Каждый час
   - Каждые 3 часа
   - Раз в день (в 03:00)
   - Раз в неделю

3. Сохраните настройки

### 7.2. Интеграция с DaData

#### Получение API-ключа
1. Зарегистрируйтесь на https://dadata.ru/
2. Пополните баланс (минимальная сумма 500₽)
3. Перейдите в личный кабинет → "API-ключи"
4. Скопируйте токен (начинается с `Token ...`)

#### Настройка на сайте
1. Перейдите в **Настройки** → **Интеграции** → **DaData**
2. Вставьте API-ключ в поле **API Token**
3. Нажмите **Проверить ключ**
   - Система сделает тестовый запрос к DaData
   - При успехе отобразится сообщение "Ключ действителен"

4. Настройте параметры:
   - **Автоматическое заполнение**: включено/выключено
   - **Задержка перед запросом** (debounce): 500-2000 мс
   - **Минимальная длина ИНН**: 10 (для юрлиц) или 12 (для ИП)

5. Сохраните настройки

#### Тестирование
1. Откройте форму регистрации оптового клиента
2. Введите тестовый ИНН (например, `7707083893` - Сбербанк)
3. Подождите 1-2 секунды
4. Поля должны заполниться автоматически:
   - Название организации
   - Адрес
   - Руководитель
   - Контакты

### 7.3. Настройка email-уведомлений
1. Перейдите в **Настройки** → **Email**
2. Заполните данные SMTP-сервера:
   - **SMTP хост**: например, `smtp.yandex.ru`
   - **SMTP порт**: 465 (SSL) или 587 (TLS)
   - **Логин**: ваш email
   - **Пароль**: пароль приложения (не от почты!)
   - **От кого**: имя отправителя (например, "Addis Coffee")

3. Нажмите **Отправить тестовое письмо**
4. Проверьте почту

5. Настройте шаблоны уведомлений:
   - **Новый заказ** (администратору)
   - **Подтверждение заказа** (клиенту)
   - **Изменение статуса** (клиенту)
   - **Счет на оплату** (B2B клиенту)

### 7.4. Настройка SMS-уведомлений
1. Перейдите в **Настройки** → **SMS**
2. Выберите провайдера:
   - SMS.ru
   - SMS-Prosto
   - Twilio (для международных)

3. Введите API-ключ провайдера
4. Настройте события для отправки:
   - Новый заказ
   - Статус "Отправлен"
   - Готов к выдаче

## 8. Безопасность и обслуживание

### 8.1. Управление пользователями админ-панели

#### Добавление нового администратора
1. Перейдите в **Настройки** → **Пользователи**
2. Нажмите **+ Добавить пользователя**
3. Заполните данные:
   - **Логин**: уникальное имя пользователя
   - **Email**: для восстановления пароля
   - **Пароль**: мин. 8 символов (буквы + цифры)
   - **Роль**: 
     - **Супер-админ**: полный доступ
     - **Менеджер товаров**: только товары и заказы
     - **Контент-менеджер**: только блог и страницы
     - **Оператор**: только заказы и клиенты

4. Сохраните

#### Изменение роли пользователя
1. Найдите пользователя в списке
2. Нажмите ✏️ (Редактировать)
3. Измените роль в dropdown
4. Сохраните

#### Блокировка пользователя
1. Откройте карточку пользователя
2. Переключите статус на **Заблокирован**
3. Пользователь не сможет войти в систему

### 8.2. Смена пароля
1. Кликните на свое имя в верхнем правом углу
2. Выберите **Настройки профиля**
3. Введите текущий пароль
4. Введите новый пароль дважды
5. Сохраните

**Требования к паролю:**
- Минимум 8 символов
- Хотя бы одна заглавная буква
- Хотя бы одна цифра
- Хотя бы один специальный символ (!@#$%)

### 8.3. Резервное копирование

#### Ручное создание бэкапа
1. Перейдите в **Настройки** → **Бэкапы**
2. Нажмите **Создать бэкап**
3. Выберите компоненты:
   - База данных (товары, заказы, клиенты)
   - Файлы (изображения, документы)
   - Настройки дизайна

4. Нажмите **Создать**
5. Дождитесь завершения
6. Скачайте архив на локальный компьютер

#### Автоматические бэкапы
1. Включите переключатель **Автоматическое резервное копирование**
2. Выберите частоту:
   - Ежедневно (в 02:00)
   - Еженедельно (воскресенье в 03:00)
   - Ежемесячно (1 число в 04:00)

3. Укажите место хранения:
   - Локально на сервере
   - Облако (Яндекс.Диск, Google Drive)
   - FTP-сервер

4. Настройте срок хранения:
   - Хранить последние 7 бэкапов
   - Хранить бэкапы за 30 дней
   - Хранить бессрочно

### 8.4. Журнал действий (Audit Log)
1. Перейдите в **Настройки** → **Журнал**
2. Просматривайте историю действий:
   - Кто выполнил действие
   - Какое действие выполнено
   - Когда произошло
   - IP-адрес исполнителя

3. Используйте фильтры:
   - По пользователю
   - По типу действия
   - По дате

4. Экспорт журнала:
   - Нажмите **Экспорт в CSV**
   - Откройте файл в Excel

### 8.5. Мониторинг производительности
1. Перейдите в **Настройки** → **Мониторинг**
2. Проверьте метрики:
   - **Время загрузки страниц**: должно быть < 3 сек
   - **Количество ошибок**: должно быть 0
   - **Использование памяти**: норма до 80%
   - **Свободное место на диске**: минимум 20%

3. При превышении пороговых значений:
   - Очистите кэш (кнопка **Очистить кэш**)
   - Удалите старые бэкапы
   - Оптимизируйте изображения

### 8.6. Обновление системы
1. Перейдите в **Настройки** → **Обновления**
2. Проверьте доступные обновления:
   - Версия ядра
   - Версии компонентов
   - Патчи безопасности

3. Перед обновлением:
   - **Создайте полную резервную копию**
   - Проверьте совместимость плагинов

4. Нажмите **Обновить**
5. Дождитесь завершения (не закрывайте браузер!)
6. Проверьте работоспособность сайта
