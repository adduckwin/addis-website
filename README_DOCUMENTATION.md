# Документация проекта Addis Coffee

## Содержание

Этот архив содержит полную техническую документацию для миграции сайта Addis Coffee на платформу 1С-Битрикс: Управление сайтом.

---

## Файлы документации

### 1. TECHNICAL_DOCUMENTATION.md
**Полная техническая документация сайта**

Содержит:
- Общую информацию о проекте
- Структуру страниц и маршрутов
- Спецификацию инфоблоков (products, blog, orders)
- Структуру групп пользователей
- Описание компонентов Битрикс
- Спецификацию API интеграций (1С, доставка, платежи)
- Бонусную программу
- Оптовые скидки
- SEO и метаданные
- Чек-лист миграции

### 2. BITRIX_IMPORT.sql
**SQL-скрипты для создания структуры БД**

Содержит:
- Создание типов инфоблоков
- Создание свойств инфоблоков
- Создание пользовательских полей (UF_)
- Создание групп пользователей
- Настройку статусов заказов
- Настройку служб доставки
- Настройку платёжных систем

### 3. BITRIX_INSTALL_GUIDE.md
**Руководство по установке и настройке**

Содержит:
- Требования к серверу
- Пошаговую установку Битрикс
- Настройку инфоблоков
- Импорт данных
- Настройку компонентов
- Интеграции с внешними системами
- Команды для обслуживания

### 4. products_import.json
**Данные товаров для импорта**

Содержит 50 товаров в формате JSON:
- Эспрессо-смеси (15 позиций)
- Фильтр-кофе (15 позиций)
- Дрип-пакеты (10 позиций)
- Все характеристики, цены, оптовые цены

### 5. blog_import.json
**Данные блога для импорта**

Содержит 15 статей в формате JSON:
- Бизнес-статьи
- Образовательные материалы
- Советы и лайфхаки
- Тренды индустрии

### 6. bitrix_components/
**PHP-компоненты для Битрикс**

Содержит:
- `apply_wholesale_discount.php` — компонент оптовых скидок
- `event_handlers.php` — обработчики событий (бонусы, 1С, верификация)
- `api_delivery.php` — API интеграция с логистическими службами

## Структура проекта

### Frontend (текущий)
```
app/
├── src/
│   ├── components/ui/      # 30+ UI компонентов (shadcn/ui)
│   ├── components/cart/    # Компоненты корзины
│   ├── pages/              # 15 страниц
│   ├── sections/           # 10 секций главной
│   ├── data/               # Статические данные
│   ├── store/              # Zustand store
│   └── types/              # TypeScript типы
├── public/                 # Статические файлы
└── dist/                   # Сборка
```

### Backend (миграция на Битрикс)
```
bitrix/
├── admin/                  # Админ-панель
├── components/             # Компоненты
├── modules/                # Модули
├── php_interface/          # Обработчики событий
├── local/
│   ├── components/         # Кастомные компоненты
│   ├── api/                # API endpoints
│   └── templates/          # Шаблоны
└── upload/                 # Загруженные файлы
```

## Инфоблоки (IBLOCK)

### 1. Товары (products)
- **Тип**: catalog
- **Количество**: 50 элементов
- **Разделы**: Эспрессо, Фильтр, Дрип-пакеты
- **Свойства**: 16 свойств (цены, характеристики, теги)

### 2. Блог (blog)
- **Тип**: content
- **Количество**: 15 элементов
- **Свойства**: категория, автор, теги, время чтения

### 3. Заказы (orders)
- **Тип**: sale
- **Статусы**: 7 статусов (pending → delivered)
- **Свойства**: тип заказа, бонусы, ID 1С

## Бонусная программа

| Действие | Баллы |
|----------|-------|
| Покупка | 1 за 100 ₽ |
| Дата рождения | +50 |
| Пол | +30 |
| Телефон | +20 |

**Использование**: 1 балл = 1 ₽ (макс. 50% заказа)

